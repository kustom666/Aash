Ceci est le buffer par défaut, vous pouvez l'utiliser pour faire des éditions rapides ou des tests de coloration

Pour créer un nouveau fichier, cliquez sur l'icône représentant une page blanche dans la barre d'icônes ou dans le menu fichier.
Pour ouvrir un ou plusieurs fichiers, cliquez sur l'icône représentant un dossier dans la barre d'icônes ou dans le menu fichier.

Si vous avez trouvé un bug, merci de le signaler à cette adresse : http://kustoms.fr/projectbugs/
